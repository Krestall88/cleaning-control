## Инструкция по размещению скриншотов

### Для продакшена:
1. Создайте папку `public/manual/screenshots/` в проекте
2. Поместите все скриншоты в эту папку с названиями screenshot-001.png до screenshot-041.png
3. Скриншоты будут доступны по URL: `/manual/screenshots/screenshot-XXX.png`

### Для базы данных:
Скриншоты хранятся в файловой системе, а в базе данных хранятся только метаданные:
- Номер скриншота
- Описание
- Раздел
- Путь к файлу

### Формат хранения:
```
public/
  └── manual/
      └── screenshots/
          ├── screenshot-001.png
          ├── screenshot-002.png
          ├── screenshot-003.png
          └── ...
```

### API для получения скриншотов:
- `GET /api/manual/screenshots` - список всех скриншотов
- `GET /api/manual/screenshots/[number]` - конкретный скриншот

### Рекомендации по созданию скриншотов:
1. Разрешение: 1920x1080 или 1280x720
2. Формат: PNG с прозрачностью или JPG
3. Размер файла: не более 500KB каждый
4. Качество: четкий текст, видимые элементы интерфейса
5. Язык: русский интерфейс
6. Данные: используйте тестовые данные, без реальной информации клиентов
