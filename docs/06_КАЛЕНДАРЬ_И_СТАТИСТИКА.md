# 6. КАЛЕНДАРЬ И СТАТИСТИКА

Единый календарь показывает все задачи по объектам: чек-листы, дополнительные задания и регулярные мероприятия. Это главный рабочий экран менеджеров и администраторов.

---

## 6.1 Кто что видит

- **Менеджер** — только свои объекты и три списка: просроченные, задачи на сегодня и предстоящие.
- **Заместитель администратора** — объекты, к которым он назначен, плюс статистику по ним.
- **Администратор** — весь пул задач, карточки объектов, срезы по менеджерам и доступ к дополнительным модальным окнам.

## 6.2 Верхняя панель управления

1. **Объект.** Выпадающий список «Все объекты» фильтрует задачи по одной площадке.
2. **Поиск.** Строка поиска ищет по названию объекта и задач.
3. **Вид.** Переключатель **День / Неделя / Месяц** меняет масштаб. Данные остаются теми же, меняется только группировка по датам.
4. **Навигация по датам.** Стрелки «← →» и кнопка **«Сегодня»** листают период.
5. **Настройки завершения (⚙️).** Кнопка открывает окно «Настройки завершения задач» для выбранного объекта. В нём администратор решает, обязателен ли фотоотчёт, комментарий или сразу оба требования при закрытии задач.

## 6.3 Карточки статистики

Под панелью управления отображаются пять показателей:

- **Всего задач** — сколько карточек попало в расписание за выбранный период.
- **Просрочено** — задачи, у которых срок уже прошёл.
- **На сегодня** — дедлайны текущего дня.
- **Выполнено сегодня** — сколько карточек закрыто за текущие сутки.
- **% выполнения** — доля закрытых задач относительно всех активных.

Карточки обновляются при любом изменении фильтров или после завершения задач.

## 6.4 Списки задач для менеджера

Если у сотрудника роль MANAGER, под статистикой выводятся два блока:

1. **Просрочено.** Красные карточки с бейджем «Просрочено». Нужно закрыть в первую очередь.
2. **На сегодня.** Голубые карточки с задачами текущего дня.

Каждая карточка содержит:
- название задачи (обычно совпадает с техкартой);
- объект и адрес;
- периодичность выполнения;
- дату и время последнего выполнения;
- кнопку **«Открыть»** и **«Выполнить»**.

## 6.5 Как закрыть задачу

1. Нажмите карточку → откроется окно с описанием и кнопкой **«Отметить выполненной»**.
2. В модальном окне укажите комментарий, добавьте фото/файл (если требуется регламентом) и подтвердите.
3. После отправки карточка переместится в «Выполнено», статистика обновится автоматически.

![screenshot-037](screenshot-037.png)
*Скриншот 37: Модальное окно завершения задачи*

> Менеджер может только отметить выполнение и оставить комментарии. Изменить задачу или её расписание могут лишь администраторы.

## 6.5 FAQ

**Можно ли создать задачу напрямую из календаря?**  
Нет. Задачи создаются в «Дополнительных заданиях» или «Отчетности по чек-листам». Календарь просто показывает их.

**Почему я вижу только свои объекты?**  
Менеджеру показываются только назначенные объекты. Для просмотра всех объектов нужны права администратора.

**Как узнать периодичность задачи?**  
Откройте карточку — под названием указан режим (ежедневно, еженедельно, разово). После завершения следующая дата ставится автоматически.

**Что делать, если данные не обновились?**  
Нажмите кнопку **«Сегодня»** или обновите страницу. Если цифры не меняются, проверьте интернет или повторно откройте раздел.
