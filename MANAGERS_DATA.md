# ДАННЫЕ МЕНЕДЖЕРОВ ДЛЯ ПРИВЯЗКИ

## ИНСТРУКЦИЯ:

1. Остановите сервер: `Ctrl+C`
2. Выполните: `npx prisma generate`
3. Запустите скрипт: `npx tsx scripts/assign-managers-full.ts`
4. Запустите сервер: `npm run dev`

## ВАЖНО:

- Скрипт автоматически найдет объекты и участки по названиям
- Привяжет менеджеров к участкам или объектам
- Покажет подробный лог выполнения

## ДАННЫЕ ИЗ СПИСКА:

Всего объектов: ~80
Менеджеров для привязки: ~15

Скрипт создан в: `scripts/assign-managers-full.ts`
